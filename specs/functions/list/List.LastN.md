# List.LastN

**Family.** F06 — table and list row trim. See `families/F06_table_row_trim.md`.

**Per-member operation.** Returns the last N elements of the input list, in source order.

**Edge cases.**
- N greater than length → input unchanged.
- N is null → treated as zero.

**Conformance.** `conformance/functions/List.LastN/`.
