# List.RemoveFirstN

**Family.** F06 — table and list row trim. See `families/F06_table_row_trim.md`.

**Per-member operation.** Returns the input list with the first N elements removed.

**Edge cases.**
- N greater than length → empty list.
- N is null → treated as zero (input unchanged).

**Conformance.** `conformance/functions/List.RemoveFirstN/`.
