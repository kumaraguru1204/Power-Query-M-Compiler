# List.Skip

**Family.** F06 — table and list row trim. See `families/F06_table_row_trim.md`.

**Per-member operation.** Returns the input list with the first N elements removed. Functionally equivalent to `List.RemoveFirstN`. (M provides both names for historical reasons.)

**Edge cases.** Same as `List.RemoveFirstN`.

**Conformance.** `conformance/functions/List.Skip/`.
