# Table.SelectRowsWithErrors

**Family.** F07 — table row filter. See `families/F07_table_row_filter.md`.

**Per-member operation.** The complement of `Table.RemoveRowsWithErrors`: returns only the rows that have at least one cell-coercion failure.

**Edge cases.** Same status caveats as `Table.RemoveRowsWithErrors`.

**Conformance.** `conformance/functions/Table.SelectRowsWithErrors/`.
