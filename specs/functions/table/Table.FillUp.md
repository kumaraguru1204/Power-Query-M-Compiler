# Table.FillUp

**Family.** F09 — table column content. See `families/F09_table_column_content.md`.

**Per-member operation.** Like `Table.FillDown` but upward: replaces null cells with the most recent non-null value below.

**Edge cases.** Symmetric to `Table.FillDown`.

**Conformance.** `conformance/functions/Table.FillUp/`.
